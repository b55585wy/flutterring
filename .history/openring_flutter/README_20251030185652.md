## 📱 运行应用

### **启动模拟器**
```bash
flutter emulators --launch Medium_Phone_API_35
```

### **运行应用**
```bash
cd openring_flutter
flutter run -d emulator-5554
```

### **热重载**
```
r - 热重载
R - 热重启
q - 退出
```
