# SmartRing 产品工作区

此目录用于存放面向产品管理的所有文档、素材和规划。核心任务包括：

- 定义产品愿景、目标用户与核心价值
- 梳理功能需求、优先级与用户故事
- 输出里程碑计划、资源评估、交付节奏
- 管理交互稿、UX 规范、原型链接
- 记录竞品分析、市场调研、数据指标

## 推荐目录结构

```
smartring/
├── vision/            # 产品愿景、市场定位、价值主张
├── requirements/      # 功能需求、用户故事、规格说明
├── roadmap/           # 里程碑、发布计划、资源排布
├── ux/                # 交互流程、页面说明、原型链接
├── research/          # 竞品、用户调研、数据分析
└── README.md          # 本说明文档
```

根据实际需要，可以逐步补充或调整子目录。

## 协作约定

- 所有文档默认使用 Markdown，便于版本控制。
- 命名遵循 `YYYYMMDD_主题.md` 或 `主题_vN.md` 形式。
- 重要决策/变更需写在 `roadmap/` 或 `requirements/` 中，并在 README 维护索引。
- 与技术文档互相关联时，请在文档中注明链接（相对路径）。
